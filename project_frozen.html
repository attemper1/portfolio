PROJECT

겨울왕국2 프로모션 반응형 홈페이지
ANAYSE | STYLEGUIDE | CODING | PUBLISHING

ANALYSE | 영화 분석

개봉일자와 주요 관객층을 고려하여 전체적인 홈페이지의 이미지를 구상.

STYLEGUIDE | 스타일가이드

겨울에 어울리는 블루 계열의 브랜드 컬러를 기반으로 한 화이트 컬러와의 조합으로 영화의 특징을 살렸다.


폰트는 영문과 한글 혼용시 가장 자연스러운 Noto Sans KR을 채택하였다.

DRAFT | 시안

XD를 이용해 랜딩 페이지, 갤러리, 예매 페이지 및 메인메뉴 시안을 제작.


반응형 홈페이지로 PC와 모바일 모두 사용자가 원하는 정보에 쉽게 접근할 수 있도록 설계.


가변형 디자인으로 어떠한 환경에서도 최적의 레이아웃을 제공한다.

CODING | 코딩

XD로 작업한 시안을 바탕으로 HTML5/JavaScript/jQuery를 이용하여 VScode로 작업.

크롬 브라우저의 개발자 도구를 활용하여 실시간으로 마진값과 이미지 사이즈를 적용.

PUBLISHING | 배포

GitHub를 활용하여 실제 모바일 기기로 UI/UX를 확인하며 수정 및 배포작업을 완료.
